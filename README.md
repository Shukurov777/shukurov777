Я — **Bahrullo Shukurov**, разработчик, который живёт между PHP и **Python** (35–40% моего стека).  
Я не просто пишу код — я создаю **живых Telegram-ботов на Python**, которые думают, отвечают и автоматизируют всё, что можно.  

- 🐍 **Python** — мой главный инструмент для Telegram-ботов (aiogram 3.x + pyTelegramBotAPI)  
- ⚡ **Backend** на PHP + Python (FastAPI / Flask где нужно)  
- 🤖 **Telegram-боты и WebApp** — от простых уведомлений до сложных бизнес-систем  
- 🗄 **Базы данных**: PostgreSQL (люблю) + MySQL  
- 🎨 **Фронтенд**: чистый HTML + CSS + JavaScript (WebApp внутри Telegram)  
- 📦 Автоматизация, JSON, REST API, веб-хуки и всё, что заставляет данные танцевать  

**Почему я особенный?**  
Потому что я сочетаю надёжный PHP с современным Python и делаю ботов, которые выглядят и работают как настоящее волшебство.
